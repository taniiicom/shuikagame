# shuikagame
a Command-Line Suika Game that runs on "sh" (shell script).

## about
shell script だけでスイカゲームを作る実験的なプロジェクトです

## run
```
git clone https://github.com/taniiicom/shuikagame
cd shuikagame
sudo sh shuikagame.sh
```
